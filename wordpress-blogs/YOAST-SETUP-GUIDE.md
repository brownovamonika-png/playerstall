# Yoast SEO Setup Guide - Step by Step

## ⚠️ IMPORTANT: These errors are from WordPress/Yoast settings, NOT the content!

The content already has everything Yoast needs. You just need to configure Yoast in WordPress.

---

## Step-by-Step Setup in WordPress

### 1. **Set Focus Keyphrase** (This fixes most errors!)

1. In WordPress, scroll down to the **Yoast SEO** box (below the editor)
2. Click on the **"Focus keyphrase"** field
3. Type exactly: `wood vs metal sports lockers`
4. Press Enter or click outside the box

**This will fix:**
- ✅ Keyphrase length
- ✅ Previously used keyphrase
- ✅ Keyphrase density
- ✅ Keyphrase in introduction
- ✅ Keyphrase in subheading

---

### 2. **Set SEO Title**

1. In the Yoast SEO box, find **"SEO title"**
2. Click "Edit snippet"
3. Paste this exactly:
   ```
   Wood vs Metal Sports Lockers: Complete 2025 Comparison Guide | PlayerStall
   ```
4. Click "Close snippet editor"

**This fixes:** ✅ Keyphrase in SEO title

---

### 3. **Set Meta Description**

1. In the Yoast SEO box, find **"Meta description"**
2. Click "Edit snippet" (same area as SEO title)
3. Paste this exactly:
   ```
   Compare wood vs metal sports lockers: durability, cost, customization, and ROI. 30+ years experience reveals which material wins long-term. Free consultation available.
   ```
4. Click "Close snippet editor"

**This fixes:** ✅ Keyphrase in meta description

---

### 4. **Set URL Slug**

1. In WordPress, look for **"Permalink"** box (usually top right, or below title)
2. Click "Edit" next to the permalink
3. Change it to:
   ```
   wood-vs-metal-sports-lockers
   ```
4. Click "OK"

**This fixes:** ✅ Keyphrase in slug

---

### 5. **Add Images with Alt Text**

**You need to upload 7 images from your gallery:**

1. Click where you want to add an image in the post
2. Click "Add Media" button
3. Upload or select from gallery
4. **IMPORTANT:** Fill in the "Alt Text" field with these exact texts:

**Image 1:**
```
Wood vs metal sports lockers side-by-side comparison showing durability differences
```

**Image 2:**
```
Wood vs metal sports lockers construction comparison showing solid birch wood durability
```

**Image 3:**
```
Metal sports locker rust damage compared to wood sports lockers showing corrosion issues
```

**Image 4:**
```
Wood vs metal sports lockers cost comparison showing long-term savings
```

**Image 5:**
```
Custom wood sports lockers with team branding compared to metal sports lockers
```

**Image 6:**
```
Wood vs metal sports lockers maintenance comparison showing easy care
```

**Image 7:**
```
Professional athletic locker room with wood vs metal sports lockers comparison
```

**This fixes:** ✅ Keyphrase in image alt attributes

## Where to Find Yoast Settings

**In WordPress Gutenberg Editor:**
- Scroll down below the content editor
- Look for the **"Yoast SEO"** panel
- It has tabs: "General", "Social", "Advanced"

**In WordPress Classic Editor:**
- Scroll down below the content editor
- Look for **"Yoast SEO"** box
- Same settings, just different layout

---

## After Setup

Once you've done all 5 steps above, refresh the Yoast SEO box. You should see:
- ✅ Green lights for most items
- ✅ Maybe 1-2 orange/yellow (those are fine)
- ✅ Overall SEO score should be 75-95/100

---

## Still Having Issues?

If Yoast still shows errors after setting the focus keyphrase:

1. **Make sure you clicked "Edit snippet"** for SEO title and meta description
2. **Make sure images are uploaded** (not just placeholders)
3. **Refresh the page** after making changes
4. **Check that the focus keyphrase is exactly:** `wood vs metal sports lockers` (no extra spaces)
