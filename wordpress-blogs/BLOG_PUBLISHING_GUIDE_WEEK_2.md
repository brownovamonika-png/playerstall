## Quick Publishing Steps (15 minutes per post)

### Step 1: Create New Post in WordPress

1. Log into WordPress Admin: `https://playerstall.com/wp-admin/`
2. Go to **Posts → Add New**
3. Switch to **Text/HTML editor** (not Visual editor)

### Step 2: Copy & Paste Content

1. Open the HTML file (`blog-3-college-sports-lockers-guide-wordpress.html` or `blog-4-football-locker-room-design-wordpress.html`)
2. Copy ALL content (Ctrl+A, Ctrl+C or Cmd+A, Cmd+C)
3. Paste into WordPress Text editor
4. Switch back to Visual editor to preview formatting

### Step 5: Post Settings

**For Each Post:**

- **Title:** Use the H1 from the HTML file
- **Permalink/Slug:** WordPress will auto-generate (verify it's keyword-rich)
- **Category:** Create or select "Buyer Guides" or "Design Guides"
- **Tags:** Add 5-7 relevant tags:
  - Post 3: `college sports lockers`, `NCAA lockers`, `athletic lockers`, `locker room design`, `college athletics`
  - Post 4: `football lockers`, `locker room design`, `athletic facilities`, `team branding`, `sports lockers`
- **Featured Image:** Upload appropriate image
  - Post 3: College locker room image
  - Post 4: Football locker room image
- **Excerpt:** Use the meta description

### Step 6: Add FAQ Schema (Yoast Premium)

**If You Have Yoast Premium:**
- Use Yoast's FAQ block in the editor
- Copy FAQs from the HTML file
- Yoast will automatically add FAQPage schema

**If You Don't Have Yoast Premium:**
- Install **Schema Pro** plugin (free alternative)
- Or use the schema markup included in the HTML files

### Step 7: Publish

- **Visibility:** Public
- **Pingbacks & Trackbacks:** Enable
- **Comments:** Enable

## Yoast SEO Troubleshooting

### Issue: "Focus keyword not in first paragraph"
**Fix:** The keyword IS in the first paragraph (check the HTML). If Yoast doesn't recognize it, try refreshing or re-saving the post.

### Issue: "No H1 found"
**Fix:** Ensure the post title is set as H1 (usually automatic). The HTML file has `<h1>` tags that WordPress should convert.

### Issue: "Readability score too low"
**Fix:** The content is already optimized for readability. If Yoast shows warnings, they're likely minor. The content follows best practices (short sentences, subheadings, etc.).

### Issue: "No internal links found"
**Fix:** The HTML includes internal links. Make sure they're formatted correctly for your WordPress site structure (update URLs if needed).
