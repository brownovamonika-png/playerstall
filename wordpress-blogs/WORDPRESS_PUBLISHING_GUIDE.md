## Step-by-Step Publishing Instructions

### Step 1: Access WordPress Admin

1. Log into your WordPress admin panel at: `https://customsportslockers.com/wp-admin/`
2. Navigate to **Posts → Add New**

### Step 2: Set Up Post Basics

#### For Blog Post 1:
- **Title:** `Wood vs Metal Sports Lockers: Complete 2025 Comparison Guide`
- **Permalink/Slug:** `wood-vs-metal-sports-lockers-comparison` (WordPress will auto-generate this)
- **Category:** Create or select "Buyer Guides" or "Comparison Guides"
- **Tags:** Add tags like: `wood lockers`, `metal lockers`, `sports lockers comparison`, `locker room design`

#### For Blog Post 2:
- **Title:** `The Complete Guide to Custom Sports Lockers: Everything Athletic Directors Need to Know`
- **Permalink/Slug:** `complete-guide-custom-sports-lockers` (WordPress will auto-generate this)
- **Category:** Create or select "Buyer Guides" or "Guides"
- **Tags:** Add tags like: `custom sports lockers`, `athletic lockers`, `locker room planning`, `sports facility design`

### Step 3: Add Meta Description (Yoast SEO or Similar Plugin)

```

```

#### Blog Post 2 Meta Description:
```
Complete guide to custom sports lockers: planning, budgeting, design, installation, and ROI. 30+ years experience. Free custom design consultation included.
```

### Step 4: Add Featured Image

**For Blog Post 1:**
- Upload an image showing wood vs metal lockers side-by-side
- **Alt Text:** `Wood vs metal sports lockers comparison showing durability and cost differences`
- **Title:** `Wood vs Metal Sports Lockers Comparison`

**For Blog Post 2:**
- Upload an image showing custom locker design process or finished custom lockers
- **Alt Text:** `Complete guide to custom sports lockers showing planning process and design options`
- **Title:** `Custom Sports Lockers Complete Guide`

### Step 5: Paste Content

1. Open the HTML file (`blog-1-wood-vs-metal-wordpress.html` or `blog-2-custom-sports-lockers-guide-wordpress.html`)
2. Copy all content (Ctrl+A, Ctrl+C or Cmd+A, Cmd+C)
3. In WordPress, switch to **Text/HTML editor** (not Visual editor) - click "Text" tab in Classic Editor or use "Code Editor" in Gutenberg
4. Paste the content
5. Switch back to Visual editor to preview formatting

**Note:** The HTML files are formatted with proper headings, lists, and links. WordPress will render them correctly.

### Step 6: Add Schema Markup

The schema markup is included at the bottom of each HTML file. You have two options:

#### Option A: Use WordPress Plugin (Recommended)
- Install **Schema Pro** or **Yoast SEO** plugin
- Use the plugin's FAQ schema feature for the FAQ sections
- The plugin will handle Article schema automatically

#### Option B: Add Manually
- Copy the `<script type="application/ld+json">` section from the HTML file
- Add it to your theme's header.php or use a plugin like "Insert Headers and Footers"
- Or use a schema plugin to insert it per-post

## Troubleshooting Common Issues

### Issue: Formatting Looks Wrong
**Solution:** Make sure you're pasting into the Text/HTML editor, not Visual editor. WordPress will convert HTML to proper formatting.

### Issue: Links Don't Work
**Solution:** Check your WordPress permalink structure. Update links to match your site's URL structure (e.g., `/sports-lockers/` vs `/products/sports-lockers/`).

### Issue: Schema Markup Not Showing
**Solution:** Use a schema plugin (Schema Pro, Yoast SEO) or add schema via theme header/footer. Test with Google's Rich Results Test tool.

### Issue: Images Not Displaying
**Solution:** Upload images to WordPress Media Library first, then insert them into the post. Update image URLs in the HTML to match WordPress media URLs.
