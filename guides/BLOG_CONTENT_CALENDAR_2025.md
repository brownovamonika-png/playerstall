## Content Creation Workflow

### For Each Blog Post:

**Day 1: Research & Planning (1 hour)**
- [ ] Keyword research (DataForSEO or Keyword Planner)
- [ ] Competitor analysis (check top 3-5 ranking pages)
- [ ] Outline creation (5-8 main sections)
- [ ] FAQ research (6-8 questions)

**Day 2: Writing (3-4 hours)**
- [ ] Write content (3,000+ words)
- [ ] Create FAQs (6-8 questions, 150-200 words each)
- [ ] Add internal links (5-8 links)
- [ ] Optimize images (3-5 images with alt tags)

**Day 3: Optimization & Review (1 hour)**
- [ ] SEO optimization (title, meta, headings)
- [ ] Brand voice review
- [ ] Fact-checking
- [ ] Final review

**Day 4: Publishing (30 min)**
- [ ] Publish to WordPress (with Yoast SEO)

- [ ] Update blog listings
- [ ] Submit to search engines

**Day 5: Promotion (30 min)**
- [ ] Social media sharing
- [ ] Internal linking from related pages
- [ ] Email newsletter (if applicable)

## WordPress Publishing (Yoast SEO)

### Using Yoast SEO Plugin:

**For Each Post:**
1. **Focus Keyword:** Enter primary keyword
2. **SEO Title:** 50-60 characters (Yoast will show green/yellow/red)
3. **Meta Description:** 150-160 characters (Yoast will show length)
4. **Readability:** Aim for green score (Yoast analyzes this)
5. **Schema:** Use Yoast's FAQ schema feature
6. **Sitemap:** Yoast auto-generates (ensure it's submitted to Search Console)

**Yoast SEO Checklist:**
- [ ] Focus keyword in title (first 60 characters)
- [ ] Focus keyword in first paragraph
- [ ] Focus keyword in URL slug
- [ ] Focus keyword in H1 (one per page)
- [ ] Meta description includes focus keyword
- [ ] Readability score: Green (60+)
- [ ] SEO score: Green (all checks passed)
- [ ] FAQ schema added (Yoast FAQ block)
- [ ] Image alt tags include keywords
