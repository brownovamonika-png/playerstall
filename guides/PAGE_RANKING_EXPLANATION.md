# Page Ranking Explanation

**Understanding Google page numbers in your ranking reports**

---

## 📄 How Google Pages Work

Google shows **10 search results per page**:

- **Page 1:** Positions 1-10
- **Page 2:** Positions 11-20
- **Page 3:** Positions 21-30
- **Page 4:** Positions 31-40
- And so on...

---

## 🎯 Why Page Numbers Matter

### Page 1 (Positions 1-10) = 🟢 EXCELLENT
- **Visibility:** Users see these without scrolling
- **Click-through rate:** Highest (most clicks)
- **Goal:** Get as many keywords here as possible

### Page 2 (Positions 11-20) = 🟡 GOOD
- **Visibility:** Users need to click "Next" or scroll
- **Click-through rate:** Medium
- **Goal:** Move these to Page 1

### Page 3+ (Positions 21+) = 🟠 NEEDS WORK
- **Visibility:** Low (most users don't go this far)
- **Click-through rate:** Low
- **Goal:** Improve to Page 1 or 2

---

## 📊 What You'll See in Reports

### Summary Statistics

Now includes **Page Distribution**:
- Page 1 (Positions 1-10): X keywords
- Page 2 (Positions 11-20): X keywords
- Page 3 (Positions 21-30): X keywords
- Page 4+ (Positions 31+): X keywords

### Ranking Table

Each keyword shows:
- **Position:** #12
- **Page:** Page 2
- **Priority:** HIGH

### Example:

| Keyword | Position | Page | Priority |
|---------|----------|------|----------|
| custom sports lockers | #5 | Page 1 | CRITICAL |
| college sports lockers | #15 | Page 2 | CRITICAL |
| football lockers | #25 | Page 3 | HIGH |

## 📋 CSV Format

The CSV now includes both position and page:

**Format:**
```
keyword_position,keyword_page
```

**Example:**
```
custom sports lockers_position,custom sports lockers_page
12,Page 2
```

This makes it easy to:
- Track page movement over time
- Create charts showing page distribution
- Identify keywords moving between pages
