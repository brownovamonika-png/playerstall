## Keyword Integration Checklist

### ✅ Required for Every Blog Post

- [ ] **Primary Keyword Selected:** Choose ONE from 🔴 CRITICAL keywords list
- [ ] **Primary Keyword in Title Tag:** Exact match or close variant (50-60 characters)
- [ ] **Primary Keyword in Meta Description:** Natural inclusion (150-160 characters)
- [ ] **Primary Keyword in H1:** Exact match or close variant
- [ ] **Primary Keyword in First Paragraph:** Natural inclusion within first 100 words
- [ ] **Primary Keyword Density:** 1-2% throughout content (natural usage, not forced)
- [ ] **Secondary Keywords:** Include 3-5 from target keywords list naturally
- [ ] **LSI Keywords:** Include related terms (see list below)

### ✅ Keyword Placement Requirements

**Title Tag (50-60 characters):**
- Format: `[Primary Keyword]: [Benefit/Year] | PlayerStall`
- Example: `Wood Sports Lockers: Complete 2025 Guide | PlayerStall`
- Must include primary keyword

**Meta Description (150-160 characters):**
- Format: `[Benefits] + [CTA] + [Primary Keyword]`
- Example: `Premium wood sports lockers Canadian-owned and operated since 1996. 30+ years experience, lifetime guarantee. Free custom design consultation.`
- Must include primary keyword naturally

**H1 Heading:**
- Format: `[Primary Keyword]: [Benefit/Year]`
- Example: `Wood Sports Lockers: Complete 2025 Buyer's Guide`
- Must include primary keyword

**First Paragraph:**
- Must include primary keyword within first 100 words
- Natural, contextual usage (not forced)
- Example: "After 30+ years manufacturing custom wood sports lockers..."

**Body Content:**
- Primary keyword: 1-2% density (natural usage)
- Secondary keywords: Include 3-5 throughout content
- Use in H2/H3 headings when natural
- Use in internal link anchor text

**Image Alt Tags:**
- Format: `[Primary/Secondary Keyword] [Description]`
- Example: `Custom wood sports lockers for collegiate football teams`
- Include relevant keywords naturally

## Keyword Density Guidelines

### Primary Keyword Density

**Target:** 1-2% of total word count

**Example for 3,000-word post:**
- 1% = 30 mentions
- 2% = 60 mentions

**Distribution:**
- Title tag: 1 mention
- Meta description: 1 mention
- H1: 1 mention
- First paragraph: 1-2 mentions
- Body content: Remaining mentions (natural, contextual)

**Don't Over-Optimize:**
- ❌ Don't repeat primary keyword in every sentence
- ❌ Don't force keyword where it doesn't fit naturally
- ✅ Use variations and synonyms
- ✅ Use secondary keywords to break up repetition

### Secondary Keywords Density

**Target:** 3-5 secondary keywords, each used 3-5 times naturally

**Example:**
- "custom sports lockers": 5-8 mentions
- "wood athletic lockers": 3-5 mentions
- "athletic lockers": 3-5 mentions
- "locker room lockers": 2-4 mentions

## Common Mistakes to Avoid

### ❌ Don't:

1. **Keyword Stuffing:** Don't repeat primary keyword excessively
   - Bad: "Our wood sports lockers are the best wood sports lockers. Wood sports lockers from PlayerStall..."
   - Good: "Our custom wood sports lockers are designed for durability. These athletic lockers feature..."

2. **Forced Keywords:** Don't force keywords where they don't fit
   - Bad: "When choosing wood sports lockers, wood sports lockers are important because wood sports lockers..."
   - Good: "When choosing lockers for your facility, wood sports lockers offer superior durability compared to metal alternatives."

3. **Ignoring Secondary Keywords:** Don't focus only on primary keyword
   - Bad: Only using "wood sports lockers" throughout
   - Good: Using "wood sports lockers", "custom sports lockers", "athletic lockers", "locker room lockers" naturally

4. **Missing LSI Keywords:** Don't forget related terms
   - Bad: Only using exact match keywords
   - Good: Using variations like "collegiate lockers", "team lockers", "wooden lockers"

### ✅ Do:

1. **Natural Integration:** Use keywords naturally in context
2. **Variation:** Use synonyms and related terms
3. **Balance:** Mix primary, secondary, and LSI keywords
4. **Relevance:** Only use keywords when they're relevant to the content
5. **User-First:** Write for users first, SEO second

## Integration Workflow

### Step 1: Select Keywords (Before Writing)

1. **Choose Primary Keyword:**
   - Prefer 🔴 CRITICAL keywords ("wood sports lockers", "custom sports lockers")
   - Consider post topic and user intent
   - Check keyword difficulty (prefer low-medium)

2. **Choose Secondary Keywords:**
   - Select 3-5 from target keywords list
   - Ensure relevance to post topic
   - Mix high and medium priority keywords

3. **Identify LSI Keywords:**
   - List related terms from LSI keywords list
   - Consider sport-specific terms if applicable
   - Consider location-specific terms if applicable

### Step 2: Write Content (During Writing)

1. **Title & Meta:**
   - Include primary keyword in title tag
   - Include primary keyword in meta description
   - Add benefits and CTA

2. **H1 & Introduction:**
   - Include primary keyword in H1
   - Include primary keyword in first paragraph
   - Natural, contextual usage

3. **Body Content:**
   - Use primary keyword 1-2% density
   - Include secondary keywords naturally
   - Include LSI keywords where relevant
   - Use in H2/H3 headings when natural

4. **Images:**
   - Include keywords in alt tags
   - Descriptive + keyword combination

### Step 3: Review & Optimize (After Writing)

1. **Keyword Check:**
   - Verify primary keyword in title, meta, H1, first paragraph
   - Count primary keyword mentions (target 1-2%)
   - Verify secondary keywords included
   - Check LSI keywords included

2. **Natural Usage Check:**
   - Read content aloud - does it sound natural?
   - Remove forced keyword usage
   - Ensure keywords fit context

3. **Final Review:**
   - All keyword requirements met
   - Content reads naturally
   - SEO optimized but user-friendly
