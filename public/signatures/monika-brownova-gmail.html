Add this signature to Gmail
  1. Click Copy signature for Gmail below (Chrome or Edge works best). Then in Gmail go to Settings → See all settings → General → scroll to Signature, click inside the editor, press Cmd+A / Ctrl+A to clear, then Cmd+V / Ctrl+V and Save changes.
  2. If the button does not copy rich HTML (some browsers), click inside the dashed preview box, press Cmd+A / Ctrl+A to select all of it (including the hidden font block), copy, then paste into Gmail.
  3. Do not use Gmail’s Insert image for the whole block; that turns it into a flat image and removes links.

Note: Email apps often ignore fonts from the website preview. Use Copy signature for Gmail so the paste includes embedded @font-face rules (Oswald + Yantramanav from Google’s static host). Gmail may still fall back to Arial on some devices; that is a limitation of email, not the site.
